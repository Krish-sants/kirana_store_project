# 🛒 Kirana Store Digital Makeover Using Python

> **Assignment 1** | Python Programming | Data Analytics Course

A set of 5 Python programs built to help a fictional kirana (grocery) store owner automate and simplify daily operations using core Python concepts.

---

## 📁 Project Structure

```
kirana_store_project/
│
├── Q1_Billing_Calculator.ipynb       # Bill generator with 5% discount logic
├── Q2_Stock_Checker.ipynb            # Real-time stock availability lookup
├── Q3_Weekly_Customer_Tracker.ipynb  # Weekly footfall analytics & insights
├── Q4_Ration_Eligibility.ipynb       # Ration scheme eligibility checker
├── Q5_Festive_Pattern.ipynb          # Diwali pyramid pattern printer
└── README.md
```

## 🚀 How to Run

### Option 1: Jupyter Notebook (Recommended)
```bash
# Install Jupyter if not already installed
pip install notebook

# Launch Jupyter
jupyter notebook
```
Open any `.ipynb` file and run cells one by one using **Shift + Enter**.

### Option 2: Google Colab
1. Go to [colab.research.google.com](https://colab.research.google.com)
2. Click **File → Upload Notebook**
3. Upload any `.ipynb` file and run it

## 🧠 Concepts Practiced

- `input()` / `print()` — user interaction
- Arithmetic & comparison operators
- `if / elif / else` — conditional branching
- `for` / `while` loops
- Lists — indexing, slicing, iteration
- String methods — `.lower()`, `.title()`, `.center()`, `.join()`
- Formatted output using f-strings
